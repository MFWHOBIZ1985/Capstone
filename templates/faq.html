Frequently Asked Questions

    Steps to create a sprint
  • On the Project page under the ticket table click "New Sprint"
  • Enter sprint details
  • Assign tickets on the sprint detail page or the ticket creation page
  • Voila, time to start the grind!

      • Steps to Create a project
  • If no projects have been created yet, you'll have a promt on the main dahsboard to create a new project. If you have created a project already, you can click on the "New Project" icon in the Navagation Bar
  • Add the basic information for you're project.
  • Create Tickets for the project
  • Create a Sprint
  • Start slaying those tickets!

      • Steps to create a Ticket
  • In the project you want to add a ticket, press the "Add Ticket" button
  • Input the information you want to add
  • In the Ticket detail page you can further add users

      • Steps to create a User
  • On the Navagation bar click "Admin Panel"
  • Select the "Users" link
  • Select "Add User"
  • Input User information

      • Steps to Delete a User
  • On the Navagation bar click "Admin Panel"
  • Select the "Users" link
  • Select the user you want to remove with the checkbox next to their name
  • On the drop down action select delete and confirm
  • Any Ticket or Project created by this user will be removed
